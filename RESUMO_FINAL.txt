🚀 COMO USAR
════════════════════════════════════════════════════════════════════════════════

PASSO 1: Extrair o arquivo
──────────────────────────────
  $ unzip url_shortener_v2.zip
  $ cd url_shortener_v2

PASSO 2: Iniciar com Docker (RECOMENDADO)
──────────────────────────────────────────
  $ docker-compose up --build
  
  ✅ Aguarde a mensagem de sucesso
  ✅ PostgreSQL: localhost:5432
  ✅ FastAPI: http://localhost:5000
  ✅ Docs: http://localhost:5000/docs

PASSO 3: Testar a API
──────────────────────────────────────────
  Ver exemplos completos em API_EXAMPLES.md dentro do ZIP
  
  Ou execute rapidamente:
  
  # Registrar
  curl -X POST "http://localhost:5000/api/v2/auth/register" \
    -H "Content-Type: application/json" \
    -d '{"username":"teste","email":"teste@test.com","password":"senha"}'
  
  # Login
  curl -X POST "http://localhost:5000/api/v2/auth/login?username=teste&password=senha"
  
  # Usar token para testar endpoints
  # Ver mais exemplos no arquivo API_EXAMPLES.md
